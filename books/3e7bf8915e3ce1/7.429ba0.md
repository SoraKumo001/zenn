# React hooksの基本

hooksは関数コンポーネントと、そこから呼び出す関数内でのみ使用可能です

- 使用したら必ず呼び出す必要があり、条件分岐などでスキップしてはならない
- 呼び出す順序や回数が変わってはならない

クラスコンポーネントと異なり関数コンポーネントは、自由にインスタンスにアクセスできません。コンポーネント内でデータが必要な場合は、正しい順序でアクセスしないと、必要なデータにたどり着くことが出来ないのです。そのため、そのとき不要なデータがあったとしても、必ず取り出す必要があります。
